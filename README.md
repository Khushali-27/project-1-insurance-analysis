# 📊 Healthcare Data Analysis & Preprocessing for Insurance Cost Prediction

## 📌 Overview
This project focuses on analyzing a healthcare insurance dataset to understand the factors affecting insurance charges. The project includes data cleaning, exploratory data analysis (EDA), feature engineering, and visualization.

---

## 🔧 Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn (for preprocessing)

---

## 📊 Key Tasks Performed
- Cleaned and preprocessed dataset using Pandas
- Performed Exploratory Data Analysis (EDA)
- Applied feature engineering (BMI categorization, encoding)
- Standardized numerical features using Scikit-learn
- Analyzed correlations between features
- Visualized data using graphs and plots

---

## 🔍 Key Insights
- Smoking is a major factor affecting insurance charges
- Age has a strong impact on insurance cost
- BMI also contributes to higher charges

---


## 🚀 Conclusion
This project helped in understanding how data preprocessing and analysis play an important role before building machine learning models.
